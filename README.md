# md5-cli [![v](https://img.shields.io/npm/v/md5-cli2.svg)](https://www.npmjs.com/package/md5-cli2) 

## Install

```
$ npm install -g md5-cli2 
```

## Usage

```
md5 [-s,--string string] [files ...]
```

## CLI
```shell
$ md5 -s "oh my god"
$ md5 --string "oh my god"
$ md5 fixtures/a.json fixtures/b.json
```
